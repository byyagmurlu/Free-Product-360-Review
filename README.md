# 360° Ürün Görüntüleyici (Product Converter)

Bu proje, bir dizi sıralı ürün görselini (360 derece çekimler) web sitenizde etkileşimli, döndürülebilir bir 3D görüntüleyiciye dönüştürmenizi sağlayan bir araçtır.

## Özellikler

*   **Sürükle ve Bırak:** Görsellerinizi kolayca yükleyin.
*   **Canlı Önizleme:** Ayarları değiştirdikçe sonucu anında görün.
*   **Atalet (Inertia):** Doğal döndürme hissi.
*   **Zoom ve Pan:** Görsele yaklaşma ve sürükleme desteği.
*   **Gelişmiş Kontroller:** İleri/Geri ok tuşları ve Tam Ekran modu.
*   **WordPress Uyumluluğu:** Tek tıkla WordPress sitenize gömebileceğiniz hazır kod üretir.
*   **Smart URL:** Sunucudaki görselleri otomatik algılar.

## Nasıl Kullanılır?

1.  `index.html` dosyasını tarayıcınızda açın.
2.  Bilgisayarınızdan görselleri seçin veya sunucu URL'si girin.
3.  Hassasiyet, sürtünme gibi ayarları yapın.
4.  "Kodu Oluştur" butonuna basarak HTML kodunu kopyalayın.
5.  Web sitenize yapıştırın.

## Kurulum (Geliştiriciler İçin)

Herhangi bir kuruluma gerek yoktur. Sadece dosyaları bir sunucuya yüklemeniz veya yerel olarak çalıştırmanız yeterlidir.
